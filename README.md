## This is a demo for realtime chart

# Setup

## Development tools

- RVM (Ruby Version Manager)
- ruby 3.2.2
- Rails 7.0.7
- Bundler 2.4.15
- MySQL 8.0.33
- Redis 7.0.4

## Before run

- setup mysql
```shell
    mysql_tzinfo_to_sql /usr/share/zoneinfo | mysql -u root -p mysql
```

- setup db
```shell
    rails db:create
    rails db:migrate
```
